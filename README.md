# 農務小幫手（Farm Assistant）

從「開始種植」到「賣出好價」全程牽著你走的智慧農務系統。
針對**南投縣信義鄉・高冷地溫室介質栽培**、主力作物**牛番茄與彩椒**，並可擴充其他作物。

> 目標使用者：全人工施作、無深厚種植經驗，希望能**完全依靠系統照表操課**。
> 現階段：**資料收集與系統設計**（下一步進入雛形開發）。

---

## 這個系統要解決什麼

| 你的需求 | 系統怎麼幫 |
|---|---|
| 我沒經驗，不知道怎麼種 | 選作物→輸入株數，自動算用量並**每天告訴你今天做什麼、怎麼做** |
| 不知道植株出狀況了沒 | 看到徵狀→系統判斷原因與處置（缺素/病蟲害/生理障礙）|
| 怕土壤壞掉、病蟲害 | 顧好 EC/pH、養好菌、根圈健康，**預防勝於治療** |
| 想賣好價、不要爛價 | 用歷史行情估**爛價期/旺季**，卡你高冷地的**夏季產季空窗** |
| 想穩定供貨不爆量 | **分批（非同步）定植**，拉長採收、分散價格風險 |

## 🎯 信義鄉實戰策略（一句話總結）

> 信義鄉是**中部山區高冷地**，能在**平地缺貨的夏季**供應彩椒/番茄。
> 平地彩椒 11 月–隔年 5 月盛產、批發價常低於 30 元/公斤（爛價）；
> 夏季颱風季平地減產、價格飆漲（零售可破 100 元/公斤）。
> **策略＝用資料鎖定夏季高價空窗 + 分批定植分散風險 + 產銷履歷加值。**

## 開發下一步（建議順序）

1. **校正在地參數**：帶著本 repo 的資料，向**台中區農業改良場**（免費諮詢）與**信義鄉農會**確認高冷地牛番茄/彩椒的栽培曆、施肥/灌溉、病蟲害門檻。
2. **土壤檢測**：取得介質初始 EC/pH/有機質基準線。
3. **撈歷史行情**：用 `fetch_amis.py` 抓牛番茄、甜椒近 3–5 年資料，建季節模型。
4. **決定產品形式**：LINE Bot / PWA / 自架開源（farmOS/Tania）— 依「好用省事」與「開源」取捨。
5. **做 MVP**：先做「作物計算＋每日任務＋行情」三個核心功能。

> 📱 App 已完成（`app/`）：田區管理、每日待辦、每日回報動態調整、黏板 IPM、
> 用藥/施肥快選、採收分級、契作管理、市場比價（送貨決策＋收入試算）、
> 產銷履歷匯出、資料備份。**部署上線見 [docs/部署上線指南.md](docs/部署上線指南.md)**。

## ⚠️ 重要聲明

本 repo 資料多來自農業部及各區農業改良場公開資訊，數值為**示範參考值**。
因信義鄉屬高冷地，且品種、季節、介質、水質各有差異，**實際施作（尤其施肥與用藥）務必經當地農業改良場或農業推廣人員確認**。
系統定位為**決策輔助**，不能取代農業專業診斷。用藥一律依 [植物保護資訊系統](https://otserv2.acri.gov.tw/PPM/) 合法登記藥劑並遵守安全採收期。
