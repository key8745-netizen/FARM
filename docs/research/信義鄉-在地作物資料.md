# 信義鄉（南投）— 在地種植資料

> 整理日期：2026-07-20
> 目的：以使用者所在地「南投縣信義鄉」為準，收集當地氣候條件與適種作物，作為作物計算引擎的「在地校正」基礎。

## 1. 在地氣候與地理條件

| 項目 | 數值 | 對種植的意義 |
|---|---|---|
| 海拔 | 約 800–1,700 公尺（山坡地）| 屬**高冷地**，非平地栽培 |
| 氣候 | 常年雲霧、四季如春、日夜溫差大 | 蔬果香甜、嫩脆；病蟲害壓力與平地不同 |
| 地形 | 玉山山麓、坡地 | 排水、灌溉、運輸需考量 |
| 務農比例 | 全鄉約 9 成居民務農 | 農業為主要經濟來源 |

> ⚠️ **重要**：因為是高冷地，作物的**適種季節、生育天數、病蟲害**都與平地版本不同。計算引擎必須為信義鄉套用「高冷地／中高海拔」參數，而非農業部平地栽培曆。

## 2. 信義鄉適種作物清單（官方在地作物）

### 你目前種植（已是在地成熟作物 ✅）
- **蕃茄**、**彩椒** — 信義鄉農會官方在地作物清單即含此兩項，代表當地氣候適合、有產銷基礎。

### 夏季高冷蔬菜（5 月中～10 月底，全鄉約 500 公頃）
- 高麗菜、花椰菜、番茄、敏豆、豌豆、青椒等
- 特色：高冷風味、香甜嫩脆；夏季平地難種而此地可種，是**產期優勢**。

## 3. 給計算引擎的「在地校正」重點

1. **氣候帶標記**：使用者設定所在地為「信義鄉／高冷地（海拔 800–1700m）」，系統據此切換栽培曆與參數。
2. **季節優勢**：高冷地夏季可種平地難種的作物（如夏季番茄、高麗菜），系統的「適種季節判定」要用高冷版本。
3. **日夜溫差**：有利果實糖度，但需注意低溫期（冬季山區可能更冷，影響番茄/彩椒生長適溫 15–26°C）。
4. **病蟲害在地化**：高冷地常見病蟲害與平地不同，病蟲害提醒需在地校正。
5. **擴充作物優先序**：日後想擴種時，優先從在地已驗證的作物挑選（高麗菜、花椰菜、敏豆、豌豆、青梅、甜柿、葡萄…），成功率高、有產銷通路。
