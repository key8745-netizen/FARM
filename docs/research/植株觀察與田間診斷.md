# 植株觀察與田間診斷 — 設計資料

> 整理日期：2026-07-20
> 目的：讓無經驗使用者知道「每天/每週要看植株哪裡、看到什麼徵狀代表什麼、該做什麼處置」。系統把它做成「拍照/勾選徵狀 → 給出可能原因與處置」的診斷助手。
> ⚠️ **診斷須謹慎**：不同品種徵狀常有差異、田間常是**多種原因複合**，不可驟下結論。系統定位為「初步判斷＋提醒送驗」，最終以**台中區農業改良場**診斷為準。

## 1. 每日/每週觀察清單（看哪裡）

| 部位 | 看什麼 | 頻率 |
|---|---|---|
| **新葉（生長點）** | 顏色、是否皺縮/捲曲/畸形 | 每日 |
| **老葉（下位葉）** | 黃化、葉緣焦枯、斑點 | 每日 |
| **整株姿態** | 是否萎凋、矮化、徒長 | 每日 |
| **花** | 落花、花色 | 開花期每日 |
| **果實** | 裂果、果頂變色、轉色是否均勻 | 結果期每日 |
| **葉背/黏板** | 粉蝨、薊馬、蚜蟲、蟎、卵 | 每週 |
| **土壤/介質** | 乾濕、EC/pH | 每日抄錄 |

## 2. 徵狀 → 原因 → 處置 對照表

### A. 缺素／營養生理障礙

| 觀察到的徵狀 | 可能原因 | 建議處置 |
|---|---|---|
| 全株葉色變黃、葉小而硬，**下位葉較嚴重** | 缺氮 | 適量追加氮肥（勿過量致徒長）|
| **老葉**葉緣及尖端變黃焦枯、向內擴展，新葉正常；果肩維持黃色無法完整轉色 | 缺鉀 | 補鉀肥；檢查是否結果期需鉀量大 |
| **幼葉**邊緣淺綠、葉背紫色，後期葉尖葉緣枯萎、生長點附近皺縮 | 缺鈣 | 補鈣、穩定水分（缺鈣常因水分不穩導致吸收差）|
| **果頂**出現褐色小斑→擴大凹陷、果內黑褐（果實膨大期）| **尻腐病＝缺鈣**（正常果鈣 0.2–0.4%，病果僅 0.12–0.15%）| 補鈣＋**穩定灌溉勿忽乾忽濕**；摘除病果 |

> 提醒：外觀常是**多元素複合障礙**，難單靠肉眼確診 → 系統建議「土壤/葉片送驗」。

### B. 生理障礙（環境引起）

| 徵狀 | 可能原因 | 處置 |
|---|---|---|
| **落花、落果** | 溫度 <15°C 或 >35°C；病毒病；養分不足 | 調控溫度（高冷地冬季保溫、夏季遮陰通風）；查是否病毒 |
| **裂果** | 水分忽乾忽濕、結果期缺水又驟灌 | 穩定灌溉、勿驟然大量給水 |
| **葉片捲曲** | ①高溫缺水（膨壓不足）②缺鈣硼 ③摘心過早 ④**病毒病** | 先排除水分/溫度；補鈣硼；若叢生畸形疑病毒→見 C |

### C. 病蟲害（需辨識＋合法用藥）

| 徵狀 | 可能病蟲害 | 處置流程 |
|---|---|---|
| 葉片捲曲叢生、狹窄扭曲畸形、植株矮化、果小 | **病毒病**（經蚜蟲/粉蝨/汁液傳播）| 拔除病株、防治媒介昆蟲；病毒病無藥可治重在預防 |
| 黏板誘到大量小白蟲 | 粉蝨 | 達門檻才依植保系統合法用藥；可用天敵小黑花椿象 |
| 新葉/花受害、銀白條斑 | 薊馬 | 黏板監測、合法用藥 |
| 葉背細小蟲、葉面黃斑 | 蟎類 | 合法用藥、注意輪替避免抗藥性 |

**用藥一律走**：達門檻 → 查 [植物保護資訊系統](https://otserv2.acri.gov.tw/PPM/) 合法登記藥 → 依標示 → 記錄並鎖安全採收期。

## 3. 系統怎麼呈現（互動設計）

**方式一：症狀選單**
使用者點「葉子怎麼了？」→ 選「老葉黃化」→ 系統列出可能原因＋處置，並問追加問題（新葉正常嗎？）縮小範圍。

**方式二：拍照比對**
拍植株照片，比對圖庫，給初步判斷（後期可加 AI 影像辨識，資料源：作物病蟲害診斷系統）。

**方式三：定期健檢提醒**
系統每週推「植株健檢」任務，帶你逐項看新葉/老葉/花果/黏板，勾選有無異常，有異常才進診斷流程。

**共同原則**：任何診斷都附「信心度」與「建議送台中區農業改良場確認」，避免誤判導致錯誤施藥。

## 4. 田間管理日常（例行，非診斷）

- **整枝摘側芽**：單幹整枝，側芽 10 cm 摘除
- **摘老葉/病葉**：改善通風、降低病害
- **牽繩/固定**：隨植株長高持續調整
- **授粉**：溫室需人工/震動輔助
- **疏果**：第一穗著果後，調整養分分配
- **清園衛生**：病株、落果、雜草即時清除（減少病蟲源）
- **記錄**：每項施作拍照入帳
