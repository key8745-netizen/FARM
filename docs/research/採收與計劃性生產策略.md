# 採收與計劃性生產策略 — 資訊收集

> 整理日期：2026-07-20
> 目的：收集「計劃性生產、分批（非同步）栽培、產期調節、與行情結合出貨」的策略，讓系統不只教種植，還能協助**規劃何時種、何時出、怎麼賣得好**。
> 對象：信義鄉高冷地・溫室介質栽培・牛番茄與彩椒。

## 1. 分批（非同步）栽培 — 避免一次採收爆量

### 問題
一次全部定植 → 採收集中在同幾週 → 人力爆量、價格被壓低、賣不完浪費。

### 策略：分期定植（Succession Planting）
把總株數拆成幾梯次，每隔 1–2 週定植一批，讓採收期錯開、拉長、平順。

**範例（500 株拆 4 梯）**
| 梯次 | 定植日 | 初次採收（+約60天）|
|---|---|---|
| 第 1 梯 125 株 | 9/15 | 11 月中 |
| 第 2 梯 125 株 | 9/29 | 11 月底 |
| 第 3 梯 125 株 | 10/13 | 12 月中 |
| 第 4 梯 125 株 | 10/27 | 12 月底 |

→ 效果：從 11 月中一路採到隔年，**每週都有貨、人力平均、供貨穩定**。

**系統怎麼幫**：輸入總株數與想要的「每週出貨量」，系統反推該分幾梯、各梯定植日，並為每一梯各自產生一套逐日任務。

## 2. 產期調節 — 用信義鄉高冷地優勢卡好時機

- 高冷地**夏季可種平地難種的作物** → 平地缺貨時你有貨 = **價格優勢**。
- 系統結合**農產品交易行情 API** 的歷史價格，分析「哪個月採收單價最好」，回推建議定植時間（計畫性生產）。
- 避開盛產崩盤期，卡進**產季空窗**出貨。

## 3. 與市場行情結合的出貨決策

- **分級包裝**：依官方「特、優、良」三級制分級，賣相好價格高。
- **看行情出貨**：系統每日拉交易行情，提示「今日牛番茄均價 X 元、近期走勢↑/↓」，建議「現在出/再等等」。
- **多市場比價**：同一作物不同批發市場價差大，系統可比較建議送哪個市場。

行情資料源：農產品交易行情 API（[data.gov.tw/dataset/8066](https://data.gov.tw/dataset/8066)）、[交易行情站](https://amis.afa.gov.tw/)。

## 4. 穩定收入的通路策略

| 策略 | 說明 | 適合 |
|---|---|---|
| **契作（契約生產）** | 與通路商簽長期合約、固定價格 | 想要**穩定收入、抗價格波動**（越來越多通路採產地直購）|
| **共同運銷** | 透過農會/合作社集貨、分級、共同出貨 | 小農降低運銷成本、議價力 |
| **產地直購 / 宅配** | 直接對消費者或餐廳 | 高品質、有故事的產品 |
| **農會通路** | 信義鄉農會（梅子夢工廠等）在地產銷 | 在地整合、加工加值 |

## 5. 產銷履歷 — 加值與分散風險

- 產銷履歷（TAP）：結合台灣良好農業規範（TGAP）＋履歷追溯，降低食安、環境、人員風險，並能**快速釐清責任**。
- 好處：**價格加值、進入更高端通路、消費者信任**。
- 系統的「田間作業紀錄」本來就在記施肥/用藥/採收 → **可直接對接產銷履歷所需資料**，一舉兩得。
- 資源：農糧署產銷履歷專區 https://www.afa.gov.tw/cht/index.php?code=list&ids=652 、履歷系統 https://resume.afa.gov.tw/

## 6. 輪作與地力 — 長期永續

- 番茄、彩椒同屬**茄科**，連作易累積土傳病害（青枯病、萎凋病等）。
- **輪作建議**：與非茄科作物（如豆科敏豆/豌豆、十字花科高麗菜/花椰菜——皆信義鄉在地作物）輪替，養地、斷病源。
- 系統可記錄每塊地的種植史，**提醒避免連作**、建議下一輪種什麼。

## 7. 系統可提供的「生產計劃」功能總結

1. **分梯規劃器**：目標每週出貨量 → 算出分幾梯、各梯定植日。
2. **產期建議**：結合歷史行情，建議「幾月採收最划算 → 幾月定植」。
3. **出貨提示**：每日行情 + 分級建議 + 送哪個市場。
4. **輪作提醒**：依地塊種植史，避免茄科連作。
5. **履歷對接**：田間紀錄直接產出產銷履歷資料。
