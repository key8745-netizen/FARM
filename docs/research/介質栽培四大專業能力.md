## 1. 介質特性的辨識與選擇（物理性與化學性）

### 1.1 理想介質的物理性指標
| 指標 | 理想範圍 | 意義 |
|---|---|---|
| 總體密度 | 0.3–0.6 g/ml | 太重壓根、太輕不穩 |
| 充氣孔隙度 | 10–50% | 根部呼吸的關鍵；不足易根腐 |
| 有效水分 | 至少 20% | 供水能力 |
| 總孔隙度 | 越高越好（泥炭達 85–90%）| 保水＋通氣的空間 |

### 1.2 常見介質特性對照
| 介質 | 類型 | 特性 | 角色 |
|---|---|---|---|
| **泥炭 Peatmoss** | 有機 | 總孔隙 85–90%、保水 50–70%、容氣 25–35% | 保水保肥主體 |
| **椰纖 Coco peat** | 有機 | 純天然、保水強、透氣佳、環保 | 保水主體（泥炭替代）|
| **珍珠石 Perlite** | 無機 | 本身不吸水，可帶自重 3–4 倍水於表面、助通氣 | 增加通氣排水 |
| **蛭石 Vermiculite** | 無機 | 質輕、多孔、無菌、保水保肥通氣佳 | 保水保肥＋通氣 |
| **岩棉 Rockwool** | 無機 | 專業生產常用、無菌、易控 | 專業水耕/育苗 |

### 1.3 化學性關鍵
- **pH**：影響養分有效性與微生物活性，目標約 6.0–7.0。
- **EC**：反映鹽類/養分濃度，需監測避免累積。
- **CEC（陽離子交換能力）**：保肥力指標；有機介質（泥炭、椰纖）與蛭石 CEC 較高＝較能留住養分、緩衝性佳。
- **緩衝性**：CEC 高的介質對施肥失誤容忍度高（不易「幾小時就出事」）。

### 1.4 選擇原則（系統決策邏輯）
依作物根系與需求「調配比例」：保水（泥炭/椰纖）＋通氣（珍珠石）＋保肥（蛭石），依作物、季節、灌溉方式調整配比與粒徑。系統可提供「介質配方計算器」。

## 2. 精準肥水管理與動態監測

### 2.1 番茄養液配方比例（生育期動態調整）
- **生育初期＋開花結果初期**：N : P₂O₅ : K₂O ≈ **6 : 1 : 4**
- **生育後期（結果採收期）**：N : P₂O₅ : K₂O ≈ **3 : 1 : 2**（減氮增鉀比，利果實）

### 2.2 EC 隨生育期動態調整（示範值）
| 生育期 | 目標 EC |
|---|---|
| 苗期 | 2.0 |
| 生長期 | 2.5 |
| 花期 | 2.8 |
| 果期 | 3.0 |

### 2.3 給液／排液管理（防鹽分累積的核心）
- **排放率**：滴灌量的 **120%**（即多給 20% 讓多餘鹽分隨排液流出，排液率約 20%）。
- **監測排液**：量測排液的 EC/pH，若排液 EC 明顯高於給液 → 介質鹽分累積 → 需加大淋洗。
- **A/B 桶配肥**：A 桶（鈣、硝酸鉀）與 B 桶（磷酸鉀 MKP、硫酸鎂、螯合微量元素）**分開存放**，避免鈣與磷/硫酸根沉澱。

### 2.4 調整順序（重要操作原則）
間歇式追肥時**先控 EC，達標後再調 pH**。因為根系直接接觸養液、無土壤緩衝，配方/pH/EC 一偏差，幾小時內植株就出問題 → **動態監測不可少**。

## 3. 自動化灌溉與硬體系統架設

> （你目前全人工，本節作為**日後升級**的知識儲備；系統可先支援人工記錄，硬體到位再接數據。）

### 3.1 核心組成
| 模組 | 功能 |
|---|---|
| **滴灌系統** | 管路＋滴頭直送根部；比漫灌省水 20–60% |
| **施肥機／自動養液控制系統** | 工業級 pH/EC＋溫度感測，多計量泵自動調配 EC/pH |
| **感測器** | 溫濕度、CO₂、光量子(PPFD)、葉溫、土溫、介質含水率、EC |
| **控制器／定時器** | 排程灌溉、施肥、光照、遮陰 |
| **遠端控制盒** | 手機遠端監控與操控 |
| **太陽能供電** | 戶外感測器可太陽能供電、全年運作 |

### 3.2 架設要點
- **模組化、可擴充**：先裝基本滴灌＋定時器，再逐步加感測與自動配肥。
- **感測資料上雲**：無線上傳雲端平台 → 可與本系統 API 對接，實現「動態監測 → 自動任務」。
- **分區控制**：不同梯次/作物分區給液（配合分批栽培）。

### 3.3 與本系統的整合路徑
人工記錄（現在）→ 加裝感測器讀 EC/pH/含水率（半自動）→ 施肥機自動配液（全自動）。系統的資料模型預留感測欄位，硬體到位即插即用。

## 4. 根圈環境（Rhizosphere）與病害預防

### 4.1 什麼是根圈
根圈＝根表面周圍約 **直徑 40 mm** 受根系分泌物與微生物活動直接影響的狹窄區域。**植物對病害的抵抗多發生在此**。顧好根圈＝顧好第一道防線。

### 4.2 有益菌如何保護根部
- 根圈有益菌（PGPR，如**假單胞菌、解澱粉芽孢桿菌**）對病原菌有**直接拮抗**作用。
- 更能誘發植物**系統性抗病（ISR, Induced Systemic Resistance）**＝活化植物自身防禦，全株抗病。
- 抗病番茄品種會在根圈**富集特定有益菌，形成防禦保護層**。
- 有益菌還協助抗旱、抗鹽、抗溫度等逆境。

### 4.3 打造健康根圈的做法（系統會引導）
1. **通氣**：介質充氣孔隙足（見第 1 節），根能呼吸 → 防根腐。
2. **有機質**：作為有益菌食物，養菌先養土。
3. **接種有益菌**：木黴菌、枯草桿菌、PGPR，建立保護層。
4. **穩定 EC/pH**：避免鹽害與極端酸鹼傷根與益菌。
5. **避免濫用殺菌劑**：會殺死根圈益菌，破壞保護層。
6. **抗病砧木嫁接**：從品種端強化根部抗病。

### 4.4 病害預防邏輯（預防 > 治療）
健康根圈（通氣＋有機質＋益菌＋穩定環境）→ 誘導抗病＋拮抗病原 → **從源頭降低青枯病、萎凋病、根腐病發生率**，減少用藥。
