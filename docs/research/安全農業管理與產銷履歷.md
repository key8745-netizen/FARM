# 台灣安全農業管理與產銷履歷（TGAP）— 合規設計

> 整理日期：2026-07-21
> 目的：使用者採「台灣安全農業管理」，系統的紀錄與用藥須符合 TGAP／產銷履歷（TAP）要求，能通過驗證稽核。
> 對象：信義鄉高冷地溫室・牛番茄與彩椒。

## 1. 制度概念

- **產銷履歷（TAP）**：遵守臺灣良好農業規範（TGAP），從源頭把關、揭露生產資訊、降低食安風險，可快速釐清責任。
- **TGAP**：政府依各品項生長特性制訂的作業標準，含流程圖、風險管理表、查核表及附表；涵蓋食品安全、環境衛生、永續生產。
- **驗證稽核重點**：生產流程、**用藥資料**、資材管理、**文件紀錄**是否符合規定。
- 精神：建立流程觀念、風險管理能力，**確實做好生產紀錄的誠實態度**。

## 2. 安全用藥五要點（系統用藥流程須內建）

| 要點 | 內容 | 系統如何落實 |
|---|---|---|
| **適藥** | 不用未核准/禁藥；正確診斷、合理用藥；**輪替不同作用機制**延緩抗藥性 | 用藥前導向植保系統查登記藥；記錄作用機制、提醒輪替 |
| **適期** | 配合**病蟲害監測**，發生初期及時防治，用最少量 | 黏板監測達門檻才用藥（IPM） |
| **適量** | 依植保手冊建議濃度，不任意增減 | 記錄稀釋倍數，比對標示 |
| **稀釋倍數** | 按標示之使用方法、範圍、稀釋倍數 | 用藥紀錄必填稀釋倍數與對象作物 |
| **安全採收期(PHI)** | 施藥後嚴守安全採收期才可採收 | **系統自動鎖定，未過期禁止採收並警示** |

資料源：農藥安全使用管理措施（農業部）https://www.moa.gov.tw/ws.php?id=4149

## 3. 田間紀錄簿必備欄位（供履歷稽核）

系統的「用藥/施肥/資材」紀錄應可匯出成含下列欄位的紀錄簿：

**病蟲害防治（用藥）紀錄**
- 日期、田區/批次、**作物**、**防治對象**（病蟲害名）、**藥劑名稱（含許可證字號佳）**、**稀釋倍數/使用量**、施藥方法、**安全採收期**、施藥人。

**施肥/資材紀錄**
- 日期、田區、資材名稱、用量、施用方法、施用人。

**栽培作業紀錄**
- 定植、整枝、授粉、採收（含產量）等作業與日期。

## 4. 合理化施肥

- 依**土壤檢測**結果施肥（避免過量致鹽害與環境負荷）。
- 記錄每次施肥種類與用量，對接履歷資材紀錄。
- 建議送土壤樣本至改良場或土壤調查中心做檢測校正。

## 5. 系統合規功能對應

| TGAP 要求 | 系統功能 |
|---|---|
| 用藥符合登記、稀釋倍數、安全採收期 | 用藥紀錄含對象/藥劑/倍數/PHI，PHI 自動鎖採收 |
| 病蟲害監測後才防治 | 黏板巡查任務 + IPM 門檻邏輯 |
| 完整文件紀錄 | 田區紀錄歷史 + **履歷紀錄簿匯出** |
| 合理化施肥 | 施肥紀錄 + EC/pH 監測 + 土檢建議 |
| 誠實可追溯 | 每筆紀錄含日期/田區/內容，時序完整 |
