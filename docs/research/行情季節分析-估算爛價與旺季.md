# 行情季節分析 — 用歷史價量估算爛價期與旺季

> 整理日期：2026-07-20
> 目的：用市場歷史「價格＋交易量」資料，算出哪些季節收成價差、哪些量大崩盤、哪些是高價空窗，並轉成種植與出貨決策。
> 對象：信義鄉高冷地・牛番茄與彩椒。
> ⚠️ 以下月份與價格為公開資料歸納之規律，實際仍受天氣、颱風、全國供給影響，僅供決策參考。

## 1. 關鍵發現：信義鄉高冷地的「產季空窗」優勢 ⭐

### 彩椒（甜椒）季節規律
| 期間 | 供給狀況 | 價格 |
|---|---|---|
| **11 月–隔年 5 月**（冬春）| 中南部**平地盛產**、氣候穩、產量大 | **最便宜**，批發價有時**低於 30 元/公斤**（爛價期）|
| **夏季–颱風季** | 高溫多濕易病、颱風豪雨減收，供應銳減 | **飆漲**，零售可破 **100 元/公斤** |
| 平地盛產後 | **由中部山區（高冷地）接棒**，供應至年底 | 相對有利 |

> 🎯 **這正是你的機會**：信義鄉屬中部山區高冷地，能在**平地缺貨的夏季／產季空窗**供貨，避開 11–5 月的平地爛價，卡進高價期。這是你最該用資料鎖定的策略。

### 番茄季節規律
- 小番茄鮮甜產季約 **12 月–隔年 4 月**（冬春），到貨量與氣候顯著影響價格。
- 參考近期批發均價（2026/7）：牛番茄約 31 元/公斤、聖女約 41、玉女可達 97（品種價差大）。

## 2. 從歷史資料能算出的「有價值資訊」

系統對 API 歷史資料（交易日期／均價／交易量）做統計，可產出：

| 分析 | 怎麼算 | 給你的價值 |
|---|---|---|
| **月均價曲線** | 各月歷年平均價 | 一眼看出哪月價高、哪月爛價 |
| **爛價季節** | 找均價最低的月份區間 | **避開這時採收** |
| **量大崩盤預警** | 交易量突增＋價格下跌 | 大家盛產→崩盤，別在此時出 |
| **高價空窗** | 均價高＋交易量低的月份 | **鎖定此時供貨**（產季調節目標）|
| **價格波動度** | 各月價格標準差 | 波動大＝風險高，波動小＝好預估 |
| **年度趨勢** | 逐年同月比較 | 長期漲跌、是否越來越多人搶種 |
| **量價關係** | 交易量 vs 均價相關性 | 驗證「量大價跌」的程度 |
| **最佳市場** | 各市場同期均價比較 | 送哪個市場價最好 |

## 3. 從分析到決策（完整鏈條）

```
歷史價量資料
   ↓ 統計分析
① 標出「爛價月」「崩盤量能」「高價空窗」
   ↓
② 鎖定目標採收月＝高價空窗（如夏季）
   ↓ 回推（採收 = 定植 + 生育天數約60天）
③ 算出最佳定植日
   ↓
④ 分批定植，主力對準高價期、避開爛價期
   ↓ 採收期
⑤ 每日行情 → 現在出/等 + 送哪市場 + 分級
```

## 4. 系統呈現方式

- **季節熱力圖**：12 個月 × 價格高低用顏色標，紅=爛價、綠=好價，直覺易懂。
- **「這個月能種嗎」建議**：輸入想種的作物，系統對照「定植後約 60 天採收落在哪個價格帶」，給紅/黃/綠燈。
- **崩盤警示**：偵測到某作物交易量異常放大＋價跌，推播「近期恐供過於求，審慎增產」。
- **空窗提醒**：「7–9 月為彩椒高價空窗，你高冷地有產期優勢，建議規劃此期供貨」。

## 5. 價格預測方法（由簡到繁，可分階段做）

1. **移動平均**：近 N 期均價平滑，看趨勢方向（最簡單，先做）。
2. **季節分解**：把價格拆成「趨勢＋季節＋隨機」，抓穩定的季節型態。
3. **同期比較**：與去年同期比，判斷今年偏高/偏低。
4. **（進階）機器學習**：納入天氣、颱風、到貨量等變數預測短期價格。

> 建議務實從「移動平均＋季節分解」起步，資料累積後再上機器學習。
