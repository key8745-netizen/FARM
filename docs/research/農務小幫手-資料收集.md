# 農務小幫手 — 前期資料收集

> 整理日期：2026-07-20
> 目的：為「農務小幫手」App/服務的開發，收集可用的公開資料來源、現有競品功能、以及建議的功能方向。

## 1. 可用的開放資料與 API

### 1.1 農業部農業資料開放平臺
- 網址：https://data.moa.gov.tw/ （舊網址 https://data.coa.gov.tw/）
- 台灣農業開放資料的主要入口，涵蓋產銷、價格、補助、農地等資料集。

### 1.2 農產品交易行情 API
- 資料集：https://data.gov.tw/dataset/8066 、 https://data.moa.gov.tw/open_detail.aspx?id=037
- 查詢站：農產品批發市場交易行情站 https://amis.afa.gov.tw/
- 欄位：交易日期、作物代號/名稱、市場代號/名稱、上價、中價、下價、平均價（元/公斤）、交易量（公斤）。
- 未帶查詢日期時，預設回傳近四天資料；每日約 AM 10:30 更新。
- 用途：**即時菜價 / 果價查詢、價格走勢圖、出貨時機建議**。

### 1.3 中央氣象署氣象資料開放平臺
- 網址：https://opendata.cwa.gov.tw/
- RESTful API：`https://opendata.cwa.gov.tw/api/v1/rest/datastore/{dataid}?Authorization={apikey}`，註冊 email 即可免費取得 API Key。
- Swagger 文件：https://opendata.cwa.gov.tw/dist/opendata-swagger.html
- 另有農業氣象觀測網監測系統 https://agr.cwa.gov.tw/ （長期預報、歷史資料、作物生長期估算）。
- 用途：**田間天氣預報、降雨機率、寒害/豪雨警示推播**。

### 1.4 病蟲害與用藥資訊
- 植物保護資訊系統（農業藥物試驗所）：https://otserv2.acri.gov.tw/PPM/ — 響應式設計、含合法用藥查詢。
- 作物病蟲害診斷諮詢服務（開放資料集）：https://data.gov.tw/dataset/40513
- 作物病蟲害診斷管理系統（民眾端）：https://diag.tactri.gov.tw/
- 農業病蟲害智能管理決策系統（農試所）：https://azai.tari.gov.tw/
- 用途：**病蟲害查詢/回報、安全用藥建議、AI 影像辨識的資料來源**。

### 1.5 智慧農業共通資訊平台
- 網址：https://agriinfo.tari.gov.tw/ （介紹：https://www.intelligentagri.com.tw/）
- 以 Open API 匯集 37 項農業生產管理資料、上億筆紀錄，供第三方開發者介接，免自建後端資料庫。

### 1.6 其他
- 農糧署開放資料專區：https://www.afa.gov.tw/cht/index.php?code=list&ids=365
- 農村水保署資料平台 API：https://data.ardswc.gov.tw/Data/OpenData/Api
- 政府資料開放平臺（總入口）：https://data.gov.tw/

## 2. 現有競品 / 參考 App

| App | 提供者 | 主要功能 |
|---|---|---|
| 農務e把抓 | 農業部（凌誠科技） https://agrepm.coa.gov.tw/ | 耕地管理（地籍+GPS 定位）、作物栽培曆、田間作業紀錄（拍照）、派工管理、串接產銷履歷 |
| 田邊好幫手 | 農業部 https://m.moa.gov.tw/ | 交易行情、農業氣象、農政訊息查詢 |
| 農業缺工好幫手 | 農業部 | 農業人力媒合、找工 |
| 農業部 Apps 總覽 | https://www.moa.gov.tw/theme_list.php?theme=apps | 官方農業 App 清單 |

競品觀察：官方工具功能齊全但介面偏舊、學習成本高；小農實際痛點是「快速記錄、看得懂的行情與天氣、用藥合規」。

## 3. 建議功能方向（MVP 草案）

1. **田間工作日誌**：一鍵記錄施肥/噴藥/採收，支援拍照與語音輸入。
2. **今日菜價**：串接交易行情 API，顯示關注作物的各市場均價與走勢。
3. **農用天氣**：串接氣象署 API，提供田區定位的預報與災害警示。
4. **病蟲害助手**：查詢病蟲害與合法用藥；後期可加影像辨識。
5. **栽培曆/待辦提醒**：依作物與種植日期自動排程工作提醒。
